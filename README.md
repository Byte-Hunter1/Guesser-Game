🎮 Java Guesser Game

A simple command-line guessing game built using Java.
This project demonstrates the use of object-oriented programming concepts, user input handling, and basic game logic.

The game involves a Guesser, Players, and an Umpire who manages the gameplay and determines the winner.

---

🚀 Project Overview

The Guesser Game is a console-based Java application where:

* A Guesser chooses a number.
* Multiple Players attempt to guess that number.
* An Umpire collects guesses and decides which player guessed correctly.

This project was created as part of my Java learning journey to practice:

* Java classes and objects
* User input handling
* Conditional logic
* Basic game design

---

⚙️ How the Project Was Built

1️⃣ Writing the Java Code

First, the entire logic of the game was implemented in Java, including:

`Guesser` class → Generates the number
 `Player` class → Allows players to guess
  `Umpire` class → Manages the game and compares guesses

---

 2️⃣ Compiling the Code

The program was compiled using the Java compiler:

```bash
javac GuesserGame.java
```

This converts the `.java` file into bytecode (`.class`) that can run on the Java Virtual Machine.

---

 3️⃣ Running the Program

After compilation, the game was executed using:

```bash
java GuesserGame
```

Players can then enter their guesses directly through the terminal.

---

4️⃣ Generating Project Documentation

To follow professional development practices, project documentation was generated using Javadoc.

Command used:

```bash
javadoc GuesserGame.java
```

This automatically creates HTML documentation explaining the classes and methods in the project.

---

5️⃣ Viewing the Documentation

After generating the documentation:

1. Navigate to the generated documentation folder.
2. Open the file:

```
index.html
```

3. Run it in your browser to view the full project documentation.
